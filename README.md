# 📦 SmartStock - Next-Gen Inventory Management System

![Banner](https://img.shields.io/badge/Status-Active-brightgreen)
![Version](https://img.shields.io/badge/Version-3.1.0-blue)
![UI](https://img.shields.io/badge/Design-Midnight_Aurora-7000ff)

SmartStock is a high-fidelity, professional inventory management solution designed for modern retail and industrial environments. Built with a "Midnight Aurora" cyber-industrial aesthetic, it combines powerful real-time tracking with an intuitive, high-performance user interface.

---

## ✨ Key Features

### 🚀 High-Performance Billing Terminal
- **Instant POS**: Process sales in milliseconds with an automated inventory settlement engine.
- **Tax Provisioning**: Automatic 18% GST calculation for professional compliance.
- **Order Manifest**: Real-time cart management with cinematic entry/exit animations.

### 🔔 Intelligent Alert Engine
- **Siren Notifications**: Auditory "Police Siren" alerts for critical low-stock events.
- **OS-Level Integration**: Native browser notifications to ensure you never miss a stock depletion.
- **Smart Dismissal**: User-isolated silence controls to manage active alerts.

### 📊 Mission Control Dashboard
- **Real-time KPIs**: Track global assets, risk alerts, and total revenue at a glance.
- **Live Logs**: Automated transaction history with unique SKU identifiers.
- **Analytics Visualization**: Glowing charts and status badges for effortless auditing.

### 🔐 Enterprise Architecture
- **Multi-User Isolation**: Scoped `localStorage` databases keyed by user email.
- **Industrial Asset Registry**: Advanced inventory catalog with exportable CSV audit logs.
- **Glassmorphic UI**: Premium HUD-style interface with mesh gradients and blur-heavy panels.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the Registry**
   ```bash
   git clone https://github.com/vanshDalal521/Inventry-Management.git
   cd Inventry-Management
   ```

2. **Deploy Dependencies**
   ```bash
   npm install
   ```

3. **Launch Terminal**
   ```bash
   npm run dev
   ```
